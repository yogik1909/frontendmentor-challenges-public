### What I learned

#### What I'm most proud of

I'm most proud of finishing this small project on my own — it's my first fully independent solution. I know it's far from perfect, and I'm ready for honest feedback, even if it means redoing parts of the work and learning a better approach.

#### Challenges I encountered

This was my first layout after an HTML/CSS course, so almost everything felt challenging.

The hardest part was **choosing and structuring CSS selectors** — which elements to target, how specific the rules should be, and how to avoid turning the stylesheet into a mess. The page looks close to the design, but I'm still not fully confident about *why* my selector choices work the way they do.

Example of organizing styles with variables:

```css
:root {
  --white: hsl(0, 0%, 100%);
  --color-slate-300: hsl(212, 45%, 89%);
  --color-slate-500: hsl(216, 15%, 48%);
  --color-slate-900: hsl(218, 44%, 22%);
}
```

### Continued development

#### What I'd do differently next time

- Spend more time on markup and CSS structure before writing detailed styles.
- Compare my work against the design more often while building, not only at the end.

#### Areas where I'd like help

I'd appreciate feedback on **CSS selector strategy** — when to use classes vs. element selectors, how to keep specificity under control, and how to organize styles so they're easier to maintain.
